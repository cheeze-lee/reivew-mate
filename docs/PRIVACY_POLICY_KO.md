# Privacy Policy - ReviewMate (초안)

최종 수정일: 2026-02-20

ReviewMate("확장 프로그램", "서비스")는 GitHub Pull Request 코드 리뷰를 지원하기 위한 Chrome 확장 프로그램입니다. 본 정책은 확장 프로그램이 어떤 데이터를 처리하는지와 그 목적을 설명합니다.

## 1) 수집/처리하는 데이터

확장 프로그램은 다음 정보를 처리할 수 있습니다.

- 사용자 입력 데이터
  - 채팅 질문/프롬프트
  - 사용자가 GitHub 화면에서 선택한 코드/텍스트
- GitHub 컨텍스트 데이터
  - PR 메타 정보(제목, 본문 일부, 변경 파일 목록)
  - PR diff 내용(일부 잘림 가능)
  - 변경 파일/연관 파일 원문 일부(용량 제한 적용)
- 설정 데이터(로컬 저장)
  - OpenAI API Key
  - Base URL
  - Organization ID/Project ID (선택 입력)
  - 모델/프롬프트 설정
- 대화 데이터(로컬 저장)
  - PR 단위의 최근 대화 기록 일부

## 2) 데이터 사용 목적

처리된 데이터는 다음 목적에만 사용됩니다.

- 코드 리뷰 답변 생성
- 리뷰 코멘트 초안/테스트 제안/리스크 분석 제공
- 사용자가 요청한 다이어그램(mermaid) 렌더링 및 표시

## 3) 데이터 전송

- 사용자가 질문을 전송하면, 관련 컨텍스트가 사용자가 설정한 LLM API 엔드포인트(`base_url`)로 전송됩니다.
- 기본값은 OpenAI API이지만, 사용자가 다른 호환 엔드포인트를 지정할 수 있습니다.
- GitHub 접근은 사용자의 로그인 세션 권한 범위 내에서만 이뤄집니다.

## 4) 로컬 저장과 보안

- API Key 및 설정값은 사용자의 브라우저 프로필 `chrome.storage.local`에 저장됩니다.
- 대화 기록 일부도 동일한 로컬 저장소에 저장될 수 있습니다.
- 이 저장소는 브라우저 계정 동기화나 별도 암호화를 보장하지 않을 수 있습니다.
- 공유 PC/공용 계정 환경에서는 사용을 권장하지 않습니다.

## 5) 데이터 공유/판매

- 개발자는 사용자의 데이터를 제3자에게 판매하지 않습니다.
- 다만, 사용자가 설정한 LLM API 제공자 및 GitHub와의 통신은 기능 수행을 위해 필요합니다.

## 6) 사용자 제어 권한

사용자는 다음을 직접 수행할 수 있습니다.

- 옵션 페이지에서 API Key 삭제
- 확장 프로그램 삭제를 통한 로컬 데이터 제거
- 브라우저 저장소 삭제 기능을 통한 잔여 데이터 정리

## 7) 아동 대상 서비스 여부

본 확장 프로그램은 아동을 대상으로 설계되지 않았습니다.
